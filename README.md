# Sim-test

This repository is initialized and set up for experimenting with OpenAI Codex integration.

## Setup Instructions

1. Install dependencies:  
   ```
   pip install -r requirements.txt
   ```

2. Set your OpenAI API key in your environment or directly in the script.

3. Run the sample script:  
   ```
   python codex_example.py
   ```

## Files

- `codex_example.py`: Example script for interacting with OpenAI Codex.
- `requirements.txt`: Python dependencies.
- `.gitignore`: Common files to ignore.
